📌 Student Info System (Mini Project)

This is a simple Python-based mini project that stores and displays student details. It is designed for beginners to understand basic programming concepts like variables, data types, input/output, and simple data handling.

🚀 Features
Add student details (name, age, course, etc.)
Store data using variables or basic data structures (like lists/dictionaries)
Display stored student information
Easy-to-use and beginner-friendly logic
🛠️ Technologies Used
Python (Core Concepts)
📚 Concepts Covered
Variables and Data Types (int, float, string, boolean)
Type Conversion
User Input (input() function)
🎯 Purpose

This project is created as a beginner अभ्यास (practice) to build a strong foundation in Python programming.

🔮 Future Improvements
Store data permanently using files or database
Add multiple student records
Create a simple user interface
Add update and delete options
